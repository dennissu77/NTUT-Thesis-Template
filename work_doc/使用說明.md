# NTUT 碩士論文模板 - 使用說明

## 環境安裝

### Ubuntu
```bash
sudo apt-get install texlive-full
sudo apt install make
```

### macOS
```bash
brew install texlive
brew install make
```

## 使用步驟

### 步驟一：填寫論文資訊

編輯 `ntut-labels.tex`，修改以下欄位：

| 欄位 | 說明 |
|------|------|
| 論文題目 | 中文與英文題目 |
| 學生姓名 | 中文與英文姓名 |
| 指導教授 | 中文與英文姓名（可設定多位） |
| 系所名稱 | 中文與英文系所全名 |
| 學位類別 | 碩士 / 博士 |
| 畢業年月 | 民國年與西元年月 |
| 學年度與學期 | 例如 112 學年度第 2 學期 |

### 步驟二：撰寫摘要與致謝

| 檔案 | 內容 |
|------|------|
| `page/abstract.tex` | 中文摘要（建議 500 字以內、1 頁為限） |
| `page/abstract-en.tex` | 英文摘要 |
| `page/thanks.tex` | 致謝 |

### 步驟三：撰寫章節內容

章節檔案放在 `chapter/` 資料夾中，模板已提供範例：

- `chapter/chapter1-introduction.tex` — 緒論
- `chapter/chapter2-related-work.tex` — 相關研究

**新增章節方式：**

1. 在 `chapter/` 資料夾中建立新檔案，例如 `chapter3-method.tex`
2. 在 `main.tex` 中加入：
   ```latex
   \include{chapter/chapter3-method}
   ```
3. 章節內容使用 `ZhChapter` 環境包裹：
   ```latex
   \begin{ZhChapter}
   \chapter{研究方法}

   你的內容...

   \end{ZhChapter}
   ```

### 步驟四：加入參考文獻

在 `reference.bib` 中新增 BibTeX 條目，例如：

```bibtex
@article{key,
  author  = {作者},
  title   = {標題},
  journal = {期刊名稱},
  year    = {2024},
}
```

在內文中使用 `\cite{key}` 引用。引用格式為 IEEE 樣式。

### 步驟五：放入口試委員簽名頁

將口試委員簽名後的掃描檔（PDF 格式）替換 `static-page/signpage.pdf`。

### 步驟六：插入圖片與表格

將圖片放在專案根目錄或自訂資料夾，在章節中引用：

```latex
\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.8\textwidth]{image.jpeg}
  \caption{圖片說明}
  \label{fig:example}
\end{figure}
```

表格範例：

```latex
\begin{table}[htbp]
  \centering
  \caption{表格說明}
  \label{tab:example}
  \begin{tabular}{|c|c|c|}
    \hline
    欄位1 & 欄位2 & 欄位3 \\
    \hline
    資料  & 資料  & 資料  \\
    \hline
  \end{tabular}
\end{table}
```

## 編譯方式

### 使用 Make（推薦）
```bash
make          # 編譯產出 main.pdf
make clean    # 清除暫存檔案
```

### 使用 build script
```bash
./build.sh
```

### 手動編譯
```bash
xelatex main
bibtex main
xelatex main
```

編譯需執行兩次 `xelatex` 以正確處理交叉引用與參考文獻。最終輸出為 `main.pdf`。

## 專案結構

```
NTUT-Thesis-Template/
├── main.tex              # 主文件（入口點）
├── ntut-report.cls       # 自訂文件類別（格式定義）
├── ntut-labels.tex       # 論文資訊設定（題目、姓名等）
├── Makefile              # 編譯指令
├── build.sh              # 編譯腳本
├── reference.bib         # 參考文獻資料庫
├── IEEEtran.bst          # IEEE 引用格式
├── DFKai-SB.ttf          # 標楷體字型檔
├── ntut-logo.png         # 浮水印用校徽
├── ntut-logo-with-label.png      # 封面用校徽（中文）
├── ntut-logo-with-label-eng.jpg  # 封面用校徽（英文）
├── page/                 # 結構頁面
│   ├── titlepage.tex         # 封面
│   ├── blankpage.tex         # 空白頁
│   ├── abstract.tex          # 中文摘要
│   ├── abstract-en.tex       # 英文摘要
│   ├── thanks.tex            # 致謝
│   ├── table-of-content.tex  # 目錄、圖目錄、表目錄
│   └── reference.tex         # 參考文獻設定
├── chapter/              # 論文章節
│   ├── chapter1-introduction.tex
│   └── chapter2-related-work.tex
└── static-page/          # 靜態頁面
    └── signpage.pdf          # 口試委員簽名頁
```

## 注意事項

- 中文字型使用標楷體（DFKai-SB），英文字型使用 Times New Roman
- 頁碼規則：前置頁面（摘要、目錄等）使用羅馬數字，正文章節使用阿拉伯數字
- 浮水印（校徽）從第三頁開始自動套用
- 行距預設為 1.5 倍行高
- 紙張為 A4，邊界：上 4.5cm、下 0.75cm、左右各 2.5cm
